More Accelerants & Fuels - README

Mod Introduction / 模组介绍

A mod for The Long Dark that adds new fuel options and customizes accelerant behavior. / 一款适用于《漫漫长夜》的模组，新增多种燃料选择并自定义助燃剂行为。

Features / 功能说明

1. Tinder as Fuel / 火引作为燃料

- Enable/disable using tinder (tinder paper, etc.) as fire fuel. / 可开启/关闭火引（树皮、报纸等）作为火堆燃料的功能。

- Adjust the burn time multiplier of tinder (based on the original value). / 可调节火引燃烧时间倍率（基于原版燃烧时间）。

2. Animal Fat as Fuel / 动物脂肪作为燃料

- Enable/disable using animal fat as high-efficiency fire fuel. / 可开启/关闭动物脂肪作为高效火堆燃料的功能。

- Customize the burn time per kilogram of animal fat . / 可自定义每公斤动物脂肪的燃烧时间。

- Adjust the additional heat added to the fire by burning animal fat. / 可调节动物脂肪燃烧时为火堆增加的额外温度。

3. Accelerant Probability Consumption / 助燃剂概率消耗
   
- Enable/disable probability-based consumption for accelerants. / 可开启 / 关闭助燃剂的概率消耗模式。
  
- When enabled, each ignition has a 1/X chance to consume the accelerant. / 开启后，每次点火将有 1/X 的概率消耗掉助燃剂。
  
- Customize the consumption chance from 1/1 (always consume) to 1/10 (rarely consume). / 可自定义消耗概率，范围从 1/1（必定消耗）到 1/10（极少消耗）。
